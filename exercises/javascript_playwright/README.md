# JavaScript Playwright Exercises

This folder contains the JavaScript Playwright exercise scaffold.

Each topic folder contains:
- `exercise-<topic>.md` with the exercise description
- an empty `<topic>Playwright.js` file for your page object or helper code
- an empty `test<Topic>Playwright.spec.js` file for your Playwright Test implementation

The JavaScript files are intentionally empty so the exercise solutions stay yours.

Run all JavaScript Playwright tests from the repository root:

```bash
npm run test:javascript
```

Run one topic by path after you start adding tests:

```bash
npx playwright test exercises/javascript_playwright/visibility
```
